# AI Game Master

An AI-powered procedural RPG engine that creates unique game worlds dynamically.

## Vision

AI Game Master generates:

- Worlds
- Characters
- Quests
- Dialogues
- Maps
- Enemies
- Adaptive gameplay

using a combination of:

- Java-based game simulation
- Python AI services
- Large Language Models
- Procedural generation algorithms

## Tech Stack

### Backend
- Java
- Spring Boot

### AI Layer
- Python
- FastAPI

### Frontend
- React

### Database
- PostgreSQL
