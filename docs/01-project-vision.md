# AI Game Master

## Project Vision

AI Game Master is an AI-powered procedural RPG engine that generates a unique game world for every new playthrough.

Unlike traditional games with predefined stories and maps, this system dynamically creates worlds, characters, quests, dialogue, and gameplay based on procedural generation and large language models.

The goal is to combine deterministic game systems with artificial intelligence to create immersive, replayable adventures where every decision has lasting consequences.

---

## Objectives

- Generate a completely new world for every game session.
- Create unique NPCs with persistent memories.
- Generate dynamic quests based on the current world state.
- Adapt game difficulty according to player skill.
- Maintain long-term world consistency through persistent storage.
- Separate deterministic game logic from AI-generated content using a microservice architecture.

---

## Target Users

- RPG players
- AI enthusiasts
- Researchers interested in procedural content generation
- Recruiters evaluating advanced software engineering projects

---

## Core Principles

1. Every playthrough should feel unique.
2. NPCs should remember player actions.
3. The world should evolve even when the player is inactive.
4. AI should assist the game engine rather than replace it.
5. The architecture should be modular, scalable, and maintainable.

---

## Expected Outcome

A fully playable AI-powered RPG engine capable of generating coherent worlds, intelligent NPCs, adaptive quests, and evolving gameplay using Java, Spring Boot, Python AI services, PostgreSQL, React, and modern AI models.
